Adds Jibit payment gateway to WooCommerce.

== Description ==
این افزونه درگاه پرداخت جیبیت را به فروشگاه ووکامرس شما اضافه می‌کند.

== Installation ==
۱- پوشه ‘wc-jibit-payment-gateway’ را در مسیر ‘wp-content/plugins` آپلود کنید.
۲- افزونه را از طریق منوی افزونه‌ها در پنل مدیریت وردپرس فعال کنید.
۳- از طریق تنظیمات افزونه در مسیر ‘ووکامرس > پیکربندی > پرداخت‌ها > مدیریت درگاه پرداخت جیبیت’ مقدار کد پذیرنده را وارد کنید.
